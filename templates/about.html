About Our Application

This project is made up of using Flask python framework, where the user can buy & sell the stocks. The new user will by default have 10000$ to enter to the market. To Buy and sell the stock, we included IEX cloud API_KEY to fetch the current market data. Hence, according to the current market price, the stock will be bought and the same will be done while selling the stock. The portfolio get updated automatically after the events.
